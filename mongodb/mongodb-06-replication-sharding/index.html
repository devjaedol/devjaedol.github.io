    Replica Set (ліөм ң)

    Replica SetмқҖ лҸҷмқјн•ң лҚ°мқҙн„°лҘј м—¬лҹ¬ л…ёл“ңм—җ ліөм ңн•ҳм—¬ кі к°Җмҡ©м„ұкіј мқҪкё° л¶„мӮ°мқ„ м ңкіөн•©лӢҲлӢӨ.

    кө¬мЎ°

    Primary (мқҪкё°/м“°кё°) в”Җв”ҖвҶ’ Secondary 1 (мқҪкё°)
                    в””в”Җв”ҖвҶ’ Secondary 2 (мқҪкё°)
                    в””в”Җв”ҖвҶ’ Arbiter (нҲ¬н‘ңл§Ң, лҚ°мқҙн„° м—ҶмқҢ, м„ нғқм Ғ)

    н•өмӢ¬ к°ңл…җ

    кө¬м„ұ мҡ”мҶҢ м„ӨлӘ…
    Primary лӘЁл“  м“°кё°лҘј мІҳлҰ¬н•ҳлҠ” мң мқјн•ң л…ёл“ң
    Secondary Primaryмқҳ лҚ°мқҙн„°лҘј ліөм ң, мқҪкё° к°ҖлҠҘ
    Arbiter нҲ¬н‘ңм—җл§Ң м°ём—¬ (лҚ°мқҙн„° м ҖмһҘ м•Ҳн•Ё, м§қмҲҳ л…ёл“ң мӢң мӮ¬мҡ©)
    Oplog ліөм ңм—җ мӮ¬мҡ©лҗҳлҠ” мһ‘м—… лЎңк·ё (capped collection)
    Election Primary мһҘм•  мӢң мһҗлҸҷ м„ м¶ң

    Replica Set м„Өм •

    мҙҲкё°нҷ”

    // mongoshм—җм„ң мӢӨн–ү (Primaryк°Җ лҗ  л…ёл“ң)
rs.initiate({
    _id: "myReplicaSet",
    members: [
        { _id: 0, host: "mongo1:27017" },
        { _id: 1, host: "mongo2:27017" },
        { _id: 2, host: "mongo3:27017" }
    ]
})

    мғҒнғң нҷ•мқё

    rs.status()           // Replica Set мғҒнғң
rs.isMaster()         // нҳ„мһ¬ л…ёл“ң м—ӯн• 
rs.conf()             // м„Өм • нҷ•мқё
rs.printReplicationInfo()    // Oplog м •ліҙ
rs.printSecondaryReplicationInfo()  // ліөм ң м§Җм—°

    л©ӨлІ„ кҙҖлҰ¬

    // л©ӨлІ„ м¶”к°Җ
rs.add("mongo4:27017")

// Arbiter м¶”к°Җ
rs.addArb("arbiter1:27017")

// л©ӨлІ„ м ңкұ°
rs.remove("mongo4:27017")

// мҡ°м„ мҲңмң„ ліҖкІҪ (лҶ’мқ„мҲҳлЎқ Primary м„ м¶ң мҡ°м„ )
cfg = rs.conf()
cfg.members[1].priority = 2
rs.reconfig(cfg)

    Read Preference (мқҪкё° л¶„мӮ°)

    лӘЁл“ң м„ӨлӘ…
    primary Primaryм—җм„ңл§Ң мқҪкё° (кё°ліёк°’, мөңмӢ  лҚ°мқҙн„° ліҙмһҘ)
    primaryPreferred Primary мҡ°м„ , л¶Ҳк°Җ мӢң Secondary
    secondary Secondaryм—җм„ңл§Ң мқҪкё°
    secondaryPreferred Secondary мҡ°м„ , л¶Ҳк°Җ мӢң Primary
    nearest л„ӨнҠёмӣҢнҒ¬ м§Җм—°мқҙ к°ҖмһҘ м ҒмқҖ л…ёл“ң 
    // Secondaryм—җм„ң мқҪкё°
db.users.find().readPref("secondary")

// Connection Stringм—җм„ң м§Җм •
// mongodb://mongo1,mongo2,mongo3/mydb?replicaSet=myReplicaSet&readPreference=secondaryPreferred

    Write Concern (м“°кё° ліҙмһҘ мҲҳмӨҖ)

    к°’ м„ӨлӘ…
    w: 0 нҷ•мқё м•Ҳн•Ё (Fire & Forget)
    w: 1 Primary нҷ•мқё (кё°ліёк°’)
    w: вҖңmajorityвҖқ кіјл°ҳмҲҳ л…ёл“ң нҷ•мқё (к¶ҢмһҘ)
    w: 3 3к°ң л…ёл“ң нҷ•мқё
    j: true м Җл„җл§Ғк№Ңм§Җ нҷ•мқё 
    db.users.insertOne(
    { name: "нҷҚкёёлҸҷ" },
    { writeConcern: { w: "majority", j: true, wtimeout: 5000 } }
)

    Sharding (мҲҳнҸү нҷ•мһҘ)

    ShardingмқҖ лҚ°мқҙн„°лҘј м—¬лҹ¬ м„ңлІ„(Shard)м—җ л¶„мӮ° м ҖмһҘн•ҳм—¬ мҲҳнҸү нҷ•мһҘмқ„ м ңкіөн•©лӢҲлӢӨ.

    кө¬мЎ°

    Client вҶ’ mongos (Router) вҶ’ Config Server (л©”нғҖлҚ°мқҙн„°)
                        в”ңвҶ’ Shard 1 (Replica Set)
                        в”ңвҶ’ Shard 2 (Replica Set)
                        в””вҶ’ Shard 3 (Replica Set)

    кө¬м„ұ мҡ”мҶҢ

    кө¬м„ұ мҡ”мҶҢ м„ӨлӘ…
    Shard лҚ°мқҙн„°лҘј м ҖмһҘн•ҳлҠ” л…ёл“ң (к°Ғк°Ғ Replica Set)
    mongos нҒҙлқјмқҙм–ёнҠё мҡ”мІӯмқ„ м Ғм Ҳн•ң ShardлЎң лқјмҡ°нҢ…
    Config Server нҒҙлҹ¬мҠӨн„° л©”нғҖлҚ°мқҙн„°, мІӯнҒ¬ л§Өн•‘ м •ліҙ м ҖмһҘ
    Chunk лҚ°мқҙн„° л¶„н•  лӢЁмң„ (кё°ліё 128MB)
    Shard Key лҚ°мқҙн„° л¶„л°° кё°мӨҖ н•„л“ң

    Shard Key м„ нғқ

    Shard KeyлҠ” н•ңлІҲ м„Өм •н•ҳл©ҙ ліҖкІҪмқҙ м–ҙл Өмҡ°лҜҖлЎң мӢ мӨ‘н•ҳкІҢ м„ нғқн•ҙм•ј н•©лӢҲлӢӨ.

    мўӢмқҖ Shard Key лӮҳмҒң Shard Key
    лҶ’мқҖ м№ҙл””л„җлҰ¬нӢ° лӮ®мқҖ м№ҙл””л„җлҰ¬нӢ° (м„ұлі„, мғҒнғң)
    к· л“ұн•ң л¶„нҸ¬ лӢЁмЎ° мҰқк°Җ (ObjectId, нғҖмһ„мҠӨнғ¬н”„)
    мҝјлҰ¬м—җ мһҗмЈј нҸ¬н•Ё мҝјлҰ¬м—җ мӮ¬мҡ© м•ҲлҗҳлҠ” н•„л“ң

    Shard Key м „лһө

    м „лһө мҳҲмӢң мһҘм җ лӢЁм җ
    Hashed { _id: "hashed" } к· л“ұ л¶„л°° лІ”мң„ мҝјлҰ¬ л№„нҡЁмңЁ
    Ranged { created_at: 1 } лІ”мң„ мҝјлҰ¬ нҡЁмңЁ н•«мҠӨнҢҹ к°ҖлҠҘ
    Compound { tenant_id: 1, _id: 1 } к· нҳ• мһЎнһҢ л¶„л°° м„Өкі„ ліөмһЎ

    Sharding м„Өм •

    // 1. Sharding нҷңм„ұнҷ” (mongosм—җм„ң)
sh.enableSharding("mydb")

// 2. Shard KeyлЎң мқёлҚұмҠӨ мғқм„ұ
db.orders.createIndex({ user_id: "hashed" })

// 3. м»¬л үм…ҳ мғӨл”©
sh.shardCollection("mydb.orders", { user_id: "hashed" })

// мғҒнғң нҷ•мқё
sh.status()
db.orders.getShardDistribution()

    к°ңл°ң/н…ҢмҠӨнҠё лӢЁмқј мқёмҠӨн„ҙмҠӨ
    кі к°Җмҡ©м„ұ н•„мҡ” Replica Set (3л…ёл“ң)
    мқҪкё° л¶„мӮ° Replica Set + Read Preference
